=== PHP Only Block ===

WordPress の Gutenberg エディターに PHP だけで作成されたカスタムブロックを追加するプラグインです。

== 概要 ==

PHP Only Block は、JavaScript や React を使わずに、PHP のみで WordPress のカスタムブロックを作成できるプラグインです。シンプルで軽量なブロックを簡単に追加できます。

== 特徴 ==

- **PHP のみで実装**: JavaScript や React の知識がなくてもブロックを作成可能
- **軽量**: 最小限のコードで動作
- **シンプル**: 複雑な設定不要で簡単に使用可能

== 要件 ==

- WordPress: 6.9 以上
- PHP: 8.1 以上

== インストール ==

= 手動インストール =

1. プラグインファイルを `/wp-content/plugins/php-only-block/` ディレクトリにアップロード
2. WordPress 管理画面の「プラグイン」画面からプラグインを有効化

= WordPress 管理画面からインストール =

1. WordPress 管理画面の「プラグイン」→「新規追加」に移動
2. プラグインをアップロードして有効化

== 使用方法 ==

1. ブロックエディターで新しいブロックを追加
2. 「PHP Only Block」を検索して選択
3. ブロックを挿入して使用

== 開発 ==

= セットアップ =

<pre>npm install</pre>

= ビルド =

<pre>npm run build</pre>
